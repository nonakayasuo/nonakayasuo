## 職歴要約

Python・TypeScript を軸に 7 年以上の開発経験を持つフルスタックエンジニア。フロントエンド（Next.js・Vue.js・React）、バックエンド（FastAPI・Flask）、AWS を中心としたクラウドインフラ設計・構築まで一貫して担当。

直近では大手製造業向け AI SaaS の開発において Tech Lead を務め、LangChain / LangGraph / MCP を活用した RAG システムの設計・開発を主導。フリーランスエンジニアとしても複数のプロジェクトに従事し、モバイルアプリ（Flutter / iOS）・Web アプリ・データ基盤構築など幅広い開発を経験。

AI 技術の活用、アーキテクチャ設計、チームの技術支援に強みを持ち、PoC から本番運用まで一貫して推進できる。

## 自己PR

フロントエンドからバックエンド、クラウド基盤設計・構築、AIエージェント開発まで一貫して対応できるフルスタックエンジニアです。Python・TypeScript を軸に 7 年以上の開発経験を持ち、直近では LangChain / LangGraph / MCP を活用した RAG システムの設計・開発を Tech Lead として主導しています。

技術面では、AWS（EC2・Lambda・ECS・CDK など）や Google Cloud（Cloud Run・GKE）を用いたインフラ設計・運用、Next.js・AngularJS・Vue.js によるフロントエンド実装、FastAPI・Flask によるバックエンド開発、さらに Cursor・Claude Code など最新の AI 開発ツールも積極的に活用しています。

業務においては、単に要件に応えるだけでなく、プロジェクト全体のアーキテクチャを俯瞰しながら技術的な課題を先回りして解決することを意識しています。チームの Tech Lead としてメンバーの技術支援やコードレビュー、設計判断にも携わり、開発品質と生産性の向上に貢献してきました。

また、AI・ブロックチェーン・デザインなど新しい技術領域にも積極的にキャッチアップしており、事業課題に対して最適な技術選定と総合的なソリューションを提供できる点が強みです。

## この先やってみたいこと

これまでフルスタックエンジニアおよび Tech Lead として、Web アプリケーションや SaaS の開発、社内 DX の推進など、技術的な課題解決を主導してきました。現在はエンジニアとしてのキャリアに充実感を感じている一方で、今後は技術を単なる手段としてだけでなく、「ビジネスの意思決定」に深く関与する形で活用していきたいと考えています。
